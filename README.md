# KI-Suchmaschine im Bildungsbereich – Interaktives Lernmodul

Dieses Repository enthält eine statische Lernseite, die den Einsatz von KI-gestützten Suchmaschinen im Bildungsbereich vermittelt. Das Modul richtet sich an Lehrkräfte, Lernende und Bildungseinrichtungen und kombiniert Hintergrundwissen mit interaktiven Elementen wie Perspektiven-Vergleichen, Checklisten, Quizfragen und Reflexionsnotizen.

## Inhalte

- **Lernreise**: Vier Stationen erklären Grundlagen, Einsatzszenarien, Chancen/Risiken und Verantwortungsfragen.
- **Perspektiven**: Dynamische Übersicht über Sichtweisen von Lernenden, Lehrkräften und Schulleitungen.
- **Werkzeugkoffer**: Modale Fenster mit Checkliste, Unterrichtsidee und Kompetenzraster.
- **Quiz & Reflexion**: Wissensüberprüfung und lokale Notizfunktion.

## Nutzung

1. Repository klonen oder herunterladen.
2. `index.html` in einem aktuellen Browser öffnen.
3. Optionale Bereitstellung über einen lokalen Webserver (z. B. `python -m http.server`).

Alle Interaktionen laufen vollständig im Browser; es sind keine zusätzlichen Abhängigkeiten erforderlich.
